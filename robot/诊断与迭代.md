## 你现在该做的（阶段四流程）

### 1. 看诊断报告（已跑过）

```bash
uv run python taili/scripts/analyze_run.py -e taili-baseline
```

当前基线结论：
- **Reward / episode length 在涨**（0.27→2.08，11→78 步）—— 有学习信号
- **Episode 仍偏短**（78/1000 步）—— 频繁提前终止
- **Entropy / action_std 未降** —— 策略仍偏随机，需目视确认是否抽搐

### 2. Viewer 目视策略（最重要）

```bash
uv run python taili/scripts/play.py -e taili-baseline --ckpt 499
```

在 Viewer 里观察：是否站稳、是否朝 vx=0.5 方向移动、是否关节乱抖。

### 3. TensorBoard 看曲线

```bash
tensorboard --logdir taili/logs/taili-baseline/20260529_165616
```

### 4. 根据现象改配置 → 重新训练

常见改法（按 `pipeline.md`）：

| 现象 | 改哪里 |
|------|--------|
| 抽搐、动作乱 | `rewards.py` 加 action_rate 惩罚 |
| reward 平台、不走 | 调 `RewardCfg` tracking 权重、`CommandCfg` |
| 频繁倒地 | 放宽/收紧 `TerminationCfg`，或调 PD |

改完后重新训练，用 `diff logs/taili-baseline/*/config.txt` 对比 run。
