
### 1. 默认（阶段一完整流程）

先 零力矩 500 步，再 随机位置 500 步，带 Viewer：

uv run python taili/scripts/passive_random_test.py

### 2. 仅被动下落（零力矩）

uv run python taili/scripts/passive_random_test.py --mode passive

从站立姿态释放，KP/KV=0，每步 `control_dofs_force(0)`，看重力下落是否合理。

### 3. 仅随机关节

uv run python taili/scripts/passive_random_test.py --mode random

在 URDF 关节限位内每 50 步重采样目标角，弱 PD（默认 kp=40, kd=40）跟踪，看是否剧烈抖动或发散。

### 4. 无头 + 固定步数（终端日志）

uv run python taili/scripts/passive_random_test.py --no-viewer --steps 300 --log-interval 50

### 5. 可复现随机

uv run python taili/scripts/passive_random_test.py --mode random --seed 42

### 6. 随机模式从随机初始姿态开始

uv run python taili/scripts/passive_random_test.py --mode random --random-init

---

## 常用参数

|参数|默认|含义|
|---|---|---|
|`--mode`|`both`|`passive` / `random` / `both`|
|`--steps`|`500`|每种模式的仿真步数|
|`--sim-dt`|`0.02`|控制步长 [s]|
|`--log-interval`|`100`|打印 `base_z`、`max\|dof_vel\|` 的间隔|
|`--resample-interval`|`50`|random 模式重采样目标角的间隔|
|`--kp` / `--kd`|`40` / `1`|random 模式 PD|
|`--cpu`|关|使用 CPU 后端|
|`--no-viewer`|关|不打开 Viewer|

---

## 观察什么（pipeline 目标）

终端每 N 步会打印：

- `base_z`：基座高度是否连续、是否突然弹飞或穿地
- `max|dof_vel|`：关节角速度是否爆炸式增大

passive：松手后应大致倒下或着地，不应原地高速抖动。  
random：跟踪随机目标时允许晃动，但不应数值发散、整机瞬移或持续高频抽搐。

异常时优先怀疑：碰撞体、质量/惯性、关节限位与初始姿态（Genesis 可能提示 `qpos0 exceeds joint limits`）。
